Ejemplo de Listas HTML

Listas Ordenadas

Las listas ordenadas en contenido HTML lo usamos generalmente para enumerar elementos con un orden de importancia. Ejemplo:

  1. Portales Web
  2. Seminario de Software
  3. Negocios Web

  4. Elemento Dentro de Lista

    Este es un ejemplo de parrafo dentro de una lista

    1. Item 1
    2. Item 2

Lista no Ordenadas

Una lista no ordenada no existe una relación de importancia entre los items de la lista. Solo se enuncian.

  • Opción 1
  • Opción 2
  • Opción 3
  • Opción 4
